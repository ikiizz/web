Cara membuat jaringan lokal dan bisa terhubung ke internet di Debian VirtualBox
Dipublikasikan pada 9 Agustus 2024

M. Rifki Sumarna - Kali ini saya akan membagikan tutorial cara membuat 2 adapter jaringan di VirtualBox, bisa untuk jaringan lokal, juga terhubung ke Internet.

Pertama kita hubungkan komputer Windows ke internet melalui Wi-Fi, bisa juga melalui Ethernet/LAN.

Lalu buka Virtualbox, buka pengaturan OS virtualnya, pilih Network. Arahkan adapter 1 ke Host-Only Adapter.

Beralih ke adapter 2 kita arahkan ke Bridged Adapter dan pilih adapter jaringan yang digunakan di windows.

Jalankan debian virtualnya dengan klik Start.

Login menggunakan akun root supaya tidak usah pakai perintah su untuk masuk ke super user / root.

Kita konfigurasi pengaturan jaringannya di debian dengan perintah> nano /etc/network/interfaces

Tambahkan adapter jaringannya di bawah adapter enp0s3 / Host-only. Bedanya enp0s3 dengan enp0s8 adalah jenis konfigurasinya, enp0s3 untuk host-only berarti menggunakan static dan enp0s8 untuk internet berarti menggunakan dhcp. Setelah itu simpan dengan cara CTRL+X, lalu Y dan enter.

Setelah dikonfigurasi, restart service jaringannya dengan perintah> /etc/init.d/networking restart dan pastikan OK berwarna hijau.

Gunakan perintah ip a untuk melihat konfigurasi jaringan pada OS Debian. Perhatikan IP disetiap interfaces, jika sudah ada berarti konfigurasi Jaringan host-only dan internetnya sudah berhasil.

Kita coba ping ke IPv4 DNS Google dengan perintah> ping 8.8.8.8

Coba lagi dengan perintah> ping google.com, apabila keduanya memiliki output alamat ip domain, ttl, time berarti sudah berhasil terkoneksi ke internet.

Lanjut ke konfigurasi koneksi antara client windows dengan server debian agar keduanya bisa berkomunikasi, pertama cari view network connections lalu klik open.

Klik adapter Virtualbox host-only yang digunakan di virtualbox.

Kemudian klik kanan adapter jaringannya, lalu pilih properties.

Klik Internet protocol version 4 (TCP/IPv4), lalu klik Properties.

Yang asalnya automatic/DHCP, kita ubah ke static/manual lalu isi dengan ip client yang satu segmen dengan ip server, jika sudah klik OK.

Lalu uji coba ping dari client ke server di Windows, caranya cari cmd lalu klik open dan ketik perintah> ping 30.30.30.1 sesuaikan dengan IP yang kalian gunakan di Debian virtual.

Uji ping juga dari server ke client pada server Debian, caranya ketik perintah> ping 30.30.30.2 angka terakhir disesuaikan saat kalian konfigurasi ip static/manual di windows.

Pastikan output-nya ada jawaban dari ip yang di ping, ttl, time yang berarti client & server ataupun sebaliknya sudah bisa saling berkomunikasi.

Jika tidak bisa ping berarti ada sesuatu yang memblok koneksi antar client & server, untuk mengatasi hal ini caranya cari Windows Defender Firewall lalu klik open.

Di sini kita klik turn windows defender firewall on or off yang berada disebelah kiri.

Matikan Firewall dengan cara klik turn off windows defender firewall. Jika sudah klik OK.

Itulah tadi tutorial bagaimana cara membuat 2 adapter jaringan di VirtualBox, semoga bermanfaat.
