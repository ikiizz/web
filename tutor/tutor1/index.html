Cara membuat bootable multi OS
Dipublikasikan pada 9 Agustus 2024

M. Rifki Sumarna - Pada kesempatan kali saya akan membagikan tutorial bagaimana caranya membuat bootable dengan sistem operasi bervariasi hanya dalam satu buah flashdisk. Sebagai informasi bootable adalah sebuah media penyimpanan yang di dalamnya terdapat sebuah sistem operasi seperti Windows, Linux dan lainnya.

Pertama yang kita butuhkan adalah software bernama Ventoy. Aplikasi ini dapat membuat sebuah flashdisk bisa memuat banyak Sistem Operasi, tidak seperti Rufus yang hanya berfokus pada satu sistem operasi dalam satu perangkat bootable.

Kunjungi situs ventoy.net/en/download.html lalu pilih file aplikasi Ventoy versi Windows karna di sini kita akan membuat bootable multi OS di Windows.

Di halaman ini kalian pilih Download Latest Version untuk mulai mengunduh aplikasi Ventoy.

Aplikasi Ventoy akan ter-download secara otomatis dan tunggu hingga proses download selesai.

Setelah berhasil di download, sekarang buka File Explorer di Windows kalian lalu pilih folder Downloads karna biasanya file apapun yang di download dari browser akan tersimpan di folder Downloads.

Karena file aplikasi Ventoy yang kita Download berformat ZIP, maka diharuskan untuk mengekstrak terlebih dahulu agar file aplikasi bisa digunakan karena masih terbungkus oleh ZIP.

Pilih Extract untuk mengekstrak semua file yang tersimpan di dalam file ZIP. Sebagai informasi bahwa ZIP adalah suatu format file yang berisi berbagai macam file yang digabungkan menjadi satu dan dikecilkan ukurannya atau istilahnya yakni Compress File.

Siapkan Flashdisk minimal 16GB karna konteksnya kita akan membuat bootable multi OS yang mana berarti membutuhkan beberapa file ISO seperti Windows dan Linux, tetapi tergantung ukuran file ISO-nya. Apabila merasa tidak cukup untuk diisi banyak file ISO, maka disarankan untuk menggunakan flashdisk berukuran 32GB keatas sesuai dengan kebutuhan teman-teman.

Setelah selesai diekstrak selanjutnya buka folder ventoy dan cari aplikasinya yang bernama Ventoy2Disk.exe lalu buka aplikasinya dengan cara klik dua kali pada file aplikasi dan jika ada jendela popup Yes No maka pilih Yes untuk mengonfirmasi persetujuan Administrator terhadap file yang ingin dibuka.

Setelah aplikasi Ventoy berhasil terbuka, yang pertama pastikan bahwa flashdisk teman-teman dapat terdeteksi oleh aplikasi dan pastikan juga bahwa flashdisk tersebutlah yang akan digunakan untuk membuat bootable karna pada saat proses membuat bootable, semua data-data yang tersimpan dalam flashdisk akan terhapus secara otomatis karna proses format yang dilakukan oleh Ventoy agar isi flashdisk benar-benar kosong dan hanya fokus untuk file ISO sistem operasi saja.

Sebelum mulai menginstall Ventoy pada flashdisk, klik menu Option yang berada di pojok kiri atas lalu pastikan Partition Style disesuaikan dengan tipe hard disk drive (HDD) atau Solid State Drive (SSD) pada PC/Laptop teman-teman, biasanya perangkat PC/Laptop modern yang sekarang-sekarang menggunakan tipe GPT (Guid Partition Table) dan versi jadul masih menggunakan tipe MBR (Master Boot Record).

Bedanya MBR hanya dapat memuat kapasitas hingga 2 TeraByte sedangkan GPT lebih besar kapasitasnya yakni hingga 9,4 ZettaByte.

Lalu pada Partition Configuration pilih format flashdisk yang teman-teman ingin gunakan, misalnya FAT32 maka flashdisk hanya transfer data hingga 4 GigaByte dan tidak lebih dari itu, kemudian apabila memilih exFAT NTFS maka flashdisk kalian bisa transfer data lebih dari 4 GigaByte, lalu untuk format UDF biasanya digunakan untuk Blu-ray, CD dan DVD.

Sesuai judul postingan di sini yang kita gunakan ialah flashdisk, lalu untuk file ISO sendiri karna setiap file ISO tidak sama ukurannya maka lebih baik pilih exFAT atau NTFS agar proses transfer data bisa lebih dari 4 GigaByte.

Setelah semuanya dikonfigurasi dengan benar selanjutnya kita Install Ventoy ke dalam flashdisk. Akan ada Warning atau peringatan yang menginformasikan bahwa perangkat yang digunakan untuk bootable akan diformat dan semua data yang ada di dalamnya akan terhapus. Di sini teman-teman pastikan tidak ada file penting yang tersimpan di dalam flashdisk, apabila sudah dipastikan kosong maka pilih Yes.

Akan muncul Warning atau peringatan lagi, fungsinya sangat sepele tapi penting yakni memastikan lagi bahwa tidak ada data-data penting yang tersimpan di dalam flashdisk. Karna apabila masih terdapat data-data penting yang tersimpan di flashdisk dan belum di backup maka itu suatu kecerobohan fatal yang teman-teman lakukan dan itu bukan jadi tanggung jawab penulis. Oke selanjutnya jika sudah pasti maka pilih Yes.

Proses instalasi Ventoy pada flashdisk sedang berlangsung, tunggu hingga proses instalasi selesai.

Jika ada pemberitahuan Congratulations! maka flashdisk teman-teman sudah menjadi bootable dan siap digunakan untuk instalasi sistem operasi pada PC/Laptop kalian dan juga teman-teman dapat mengisi flashdisk dengan berbagai macam file sistem operasi yang berbeda.

Teman-teman cukup meng-copy atau menyalin file ISO sistem operasi yang kalian punya ke dalam flashdisk sesuai dengan kebutuhan. Teman-teman cukup hubungkan flashdisk dengan PC/Laptop dan restart perangkat lalu masuk ke flashdisk dan nanti akan ada tampilan berbagai file ISO untuk menginstall sistem operasi.

Ini adalah tampilan GRUB Bootloader dari bootable punya saya yang telah dipasang aplikasi Ventoy, terdapat daftar sistem operasi yang siap untuk di install di PC/Laptop.

Itulah tadi tutorial bagaimana cara membuat bootable multi OS dalam satu flashdisk, semoga bermanfaat.
