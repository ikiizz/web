Belajar Mikrotik dasar dan Winbox tanpa RouterBoard
Dipublikasikan pada 9 Agustus 2024

M. Rifki Sumarna - Pada kesempatan kali ini saya akan membagikan tutorial cara konfigurasi MikroTik tanpa RouterBOARD di VirtualBox, jadi cocok buat kamu yang mau belajar MikroTik tetapi kendala biaya.

Sebelum memulai, ada bahan-bahan yang perlu kita download yaitu VirtualBox, RouterOS dan Winbox. Caranya klik link di bawah ini untuk mendownload Winbox + RouterOS dan VirtualBox jika kamu belum mempunyai ketiga Software ini.

Download Winbox + RouterOS
Download VirtualBox

Yang pertama, tambahkan mesin virtual dengan cara klik New, lalu untuk nama bebas, type Other, version Other/Unknown selanjutnya klik Next.

Bagian konfigurasi Hardware biarkan default/bawaan sistem lalu klik Next.

Bagian konfigurasi Virtual Hard disk biarkan default lalu klik Next.

Jika dirasa sudah benar, klik Finish.

Selanjutnya kita konfigurasi Network Interfaces untuk MikroTik, eth1 untuk sumber Internet & eth2 untuk Jaringan LAN, pada adapter 1 ubah menjadi Bridge Adapter.

Pada adapter 2 ubah menjadi Internal Network karena yang akan jadi client adalah windows 7 versi virtual, jadi untuk menghubungkannya harus menggunakan jaringan internal antar OS Virtual.

Klik Start untuk menjalankan mesin virtual. Jika RouterOS tidak bisa booting maka ada yang harus diubah. Matikan dulu mesin virtual untuk mengubah konfigurasinya.

Buka pengaturan mikrotik RouterOS, di bagian General, ubah version menjadi Other/Unknown (64-bit).

Jalankan kembali RouterOS Virtual dengan cara klik Start.

Untuk menginstall RouterOS Mikrotik, ketik i, lalu ketik y untuk melanjutkan proses instalasi. Setelah selesai, jangan klik enter dulu karna ada pengaturan yang harus diubah.

Keluar dari mesin virtual lalu pilih Power Off the machine (tenang saja tidak akan mengulang dari awal).

Buka pangaturan mesin virtual lalu pilih menu System, di sini kita hapus tanda centang pada bagian Optical, gunanya agar saat RouterOS booting tidak mengulang dari awal lagi, karna sebenarnya sudah berhasil terinstall dan tinggal dijalankan saja.

Klik Start dan lihat hasilnya, kita hanya tinggal login saja menggunakan user bawaan yaitu admin.

Agar konfigurasinya lebih mudah & cepat, kita gunakan Winbox.

Sebelum login, klik menu Tools di pojok kiri atas, lalu klik Legacy Mode apabila menggunakan RouterOS versi jadul mulai dari versi 6.43 ke bawah.

Lalu login ke MikroTik yang sudah terhubung dengan komputer pada menu Neighbors

Pertama hubungkan MikroTik dengan Internet, di menu sebelah kiri, klik IP lalu klik DHCP Client, klik tanda + untuk menambahkan interfaces yang terhubung ke internet.

Pada bagian Interfaces, pilih port yang tersambung ke sumber Internet. Lalu klik Apply & OK.

Selanjutnya ke menu IP lalu klik Addresses, klik tanda + untuk menambahkan interfaces/port yang akan terhubung ke client.

Pada bagian Address bebas mau IP apa saja, karna anda seorang administrator Jaringan. Pada Interfaces pilih ether2 karna ether1 sudah digunakan untuk Internet.

Selanjutnya di menu IP klik DHCP Server untuk menentukan range IP yang akan dialokasikan kepada client secara otomatis. Untuk menambahkannya klik DHCP Setup.

Bagian Interfaces, pilih ether2, selanjutnya klik next-next saja sampai OK.

Selanjutnya kita konfigurasi Firewall agar client dapat terhubung ke Internet.

Bagian Out. Interfaces pilih ether1 karna sumber Internet berada pada port ether1, tergantung port mana yang terhubung ke Internet.

Lalu pada bagian Action, ubah dari accept ke masquerade. Gunanya yaitu menyamarkan IP Public ke IP Private, maksudnya client diperbolehkan terkoneksi ke internet menggunakan IP Private, dikarenakan IP Private tidak termasuk dalam routing table router yang ada pada jaringan internet global dan hanya bisa digunakan di jaringan lokal saja.

Berdasarkan pengalaman saya, saat menggunakan jaringan Host only, tidak ada alokasi IP Address yang didistribusikan secara otomatis yang sudah dikonfigurasi pada MikrtoTik Virtual. Maka dari itu kita gunakan Internal Network, karenanya kita membutuhkan Sistem Operasi yang berperan sebagai Client yang sama-sama berjalan di mesin virtual.

Di sini saya menggunakan sistem operasi Windows 7 yang bertindak sebagai client, jika kamu belum mempunyai file ISO Windows 7 kamu bisa mendownloadnya dari situs nesabamedia.com dengan cara klik link di bawah ini.

Download file ISO Windows 7

Jika sudah, kita setting terlebih dahulu adapter jaringannya dengan memilih Internal Network, karena MikroTik yang kita gunakan bersifat virtual, agar mesin virtual satu dengan mesin virtual lainnya terhubung di VirtualBox, maka gunakanlah Internal Network.

Lakukan instalasi Windows 7 sesuai intruksi yang telah diberikan pada sistem Windows hingga selesai. Untuk jenis jaringan internet, kita pilih public saja.

Setelah proses instalasi selesai, perhatikan pada menu Taskbar, klik ikon Network yang bergambar PC + kabel LAN, pastikan ada Internet access yang menandakan Client Windows 7 sudah berhasil terkoneksi dengan Internet.

Untuk memastikannya lagi, pada menu pencarian, kita cari yang namanya View network connections.

Klik dua kali pada adapter jaringan, lalu klik Details untuk melihat konfigurasi IP Address.

Kita tes menggunakan browser jadul bawaan Windows 7 yaitu Internet Explorer dan coba telusuri google lalu buka situs google. Sampai di sini Client sudah mendapatkan IP secara otomatis dari MikroTik dan terhubung ke Internet berkat bantuan masquerade pada Firewall MikroTik agar IP Private bisa terhubung ke Internet.

Sampai di sini kita sudah berhasil menghubungkan client Windows 7 ke Internet melalui jaringan internal VirtualBox dari MikroTik RouterOS. Sekian mohon maaf bila ada banyak kekurangan, semoga bermanfaat.
